{{% notice info %}}
To use Azure services, you must first create an Azure account. You will need a valid credit card(*This card is only used for identity verification*) to create a free Azure account
{{%/notice%}}

### Steps to create an Azure account

1. Go to [Azure](https://azure.microsoft.com/free)
2. Scroll down to learn more about the free service available.
![Create An Azure Account](/images/2-Prerequiste/2.1-CreateanAzureaccount/0001-createanazureaccount.png)

3. Select **Start free**

![Create An Azure Account](/images/2-Prerequiste/2.1-CreateanAzureaccount/0002-createanazureaccount.png)

4. Sign in with **Microsoft account** (if you don't have one, you can create a free Microsoft account) or **Github account**.
5. In the **About you** page, you select your exact country or region.
- Enter **First and last name**
- Enter **your email address**
- Enter **your phone number**
- Then, select **Next**

6. On the screen **Identity verification by phone**

- Select country code
- Enter your phone number
- You have the option to text or call back to get the verification code

7. Enter the code in **Verification code** and select **Verify code**
   
- If the verification code is correct, you will be asked to enter the details of a valid credit card

8. Enter card information and select **Next**
9. Check again and select **Sign up**

![Create An Azure Account](/images/2-Prerequiste/2.1-CreateanAzureaccount/0003-createanazureaccount.png)
