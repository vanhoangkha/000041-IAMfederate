1. Go to **Azure AD tenant** management page (Firstcloudjourney)

![Create Enterprise Application](/images/3-EnterpriseApplication/0001-createenterpriseapplication.png)

2. Select **Enteprise applications** in the left menu.


![Create Enterprise Application](/images/3-EnterpriseApplication/0002-createenterpriseapplication.png)

3. Currently we do not have any connection app.

![Create Enterprise Application](/images/3-EnterpriseApplication/0003-createenterpriseapplication.png)

4. Create Enterprise Application, select **New application**

![Create Enterprise Application](/images/3-EnterpriseApplication/0004-createenterpriseapplication.png)

5. In **Browse Azure AD Gallery**

- **Cloud platforms** select **Amazon Web Service**

![Create Enterprise Application](/images/3-EnterpriseApplication/0005-createenterpriseapplication.png)

6. Select **AWS Single-Account Access**

![Create Enterprise Application](/images/3-EnterpriseApplication/0006-createenterpriseapplication.png)

7. Select **Create**

![Create Enterprise Application](/images/3-EnterpriseApplication/0007-createenterpriseapplication.png)

8. Wait a few minutes, after creating the result as shown, creating **AWS Single-Account Access** is successful

![Create Enterprise Application](/images/3-EnterpriseApplication/0008-createenterpriseapplication.png)
